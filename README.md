# Screen Recorder

> **SiC Crew Milestone Documentation Tool**

Simple screen recorder for documenting feature testing milestones.

## Purpose

Record testing sessions when trying new features in SiC App and other projects.

- Capture visual demonstrations
- Document milestone moments
- WebM format (efficient, quality)
- Quick start/stop workflow

## Tech Stack

- Python
- Tauri (Rust + TypeScript)
- WebM video format
